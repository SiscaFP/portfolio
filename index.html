VisionSort — Smart Conveyor for Package Sorting

VisionSort is an automated conveyor system designed to make package sorting faster and more accurate. Equipped with an ESP32 and HC-SR04, it scans barcodes on incoming packages and routes them to the right destination using a series of DC motor-controlled mechanisms. Each package is identified by its unique barcode, and the system ensures it ends up exactly where it needs to go, no manual sorting required. It's a simple yet effective solution for warehouses, logistics hubs, or anyone looking to streamline their package handling process.
